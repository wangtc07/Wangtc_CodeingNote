# password
```regex
^(?=.*?[A-Z])(?=.*?[a-z])(?=.*?[0-9])(?=.*?[#?!@$%^&*-]).{8,}$
```
- 此正則表達式將強制執行以下規則：
	1. 至少一個大寫[英文字母](https://en.wikipedia.org/wiki/English_alphabet)，`(?=.*?[A-Z])`
	2. 至少一個小寫英文字母， `(?=.*?[a-z])`
	3. 至少一位數， `(?=.*?[0-9])`
	4. 至少一個特殊字符， `(?=.*?[#?!@$%^&*-])`
	5. 長度至少為八`.{8,}`（帶錨）
