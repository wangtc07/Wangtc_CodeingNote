#  Angular中的重要角色
- component(組件)
	- 由css,html,ts組成
		- app.component.css
		- app.component.html
		- app.component.ts
- directive(指令)
- Pipe(管道)
- Service(服務)
- Module(模塊)
- Routing(路由)
- other
	- 注入器(Injector)  
	- 裝飾器(@Decorator)


# Angular CLI專案
- `ng new 專案名` 新建專案
- `ng s` 啟動專案
